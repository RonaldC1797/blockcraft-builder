## 1. Primeros pasos

### ¿Cómo entro a la app?

Cuando abres Blockcraft Builder por primera vez verás la pantalla de inicio de sesión.

**Si ya tienes cuenta:**
1. Escribe tu **correo electrónico**
2. Escribe tu **contraseña**
3. Toca el botón **"INICIAR SESIÓN"**

**Si es tu primera vez:**
1. Toca **"¿No tienes cuenta? Registrarse"**
2. Escribe tu **nombre**
3. Escribe tu **correo electrónico**
4. Elige una **contraseña** (mínimo 6 caracteres, debe tener al menos una mayúscula)
5. Repite la contraseña
6. Toca **"REGISTRARSE"**

> 💡 **Tip:** Guarda bien tu correo y contraseña — los necesitarás cada vez que entres a la app.

## 2. Pantalla principal

Cuando entras a la app verás tu pantalla principal con todo lo que necesitas:

### ¿Qué hay en esta pantalla?

| Elemento | Para qué sirve |
|---|---|
| **¡Hola, [tu nombre]!** | Saludo personalizado con tu nombre |
| **Bloques colocados** | Cuántos bloques has puesto en todos tus mundos |
| **Horas de juego** | Tiempo aproximado que llevas construyendo |
| **Mundos recientes** | Tus últimas construcciones guardadas |
| **Botón NUEVO MUNDO** | Para crear una construcción nueva |
| **Botón CARGAR** | Para ver todos tus mundos guardados |

### Pestañas de categoría

En la parte superior hay 4 pestañas — **Todos, Naturaleza, Construcción, Mecanismo** — que filtran tus mundos según la categoría que elegiste al crearlos.

### Barra inferior

En la parte de abajo hay 3 botones:
- 🏠 **Inicio** — regresa a la pantalla principal
- 🌍 **Mundos** — va a la lista de todos tus proyectos
- 📦 **Inventario** — muestra todos los bloques disponibles

## 3. Crear un mundo nuevo

### Paso a paso

**1.** Toca el botón **"NUEVO MUNDO"** en la pantalla principal

**2.** Escribe un **nombre** para tu mundo — por ejemplo: *"Mi castillo"*, *"Casa en el árbol"*, *"Nave espacial"*

> ⚠️ El nombre debe tener al menos 3 letras y solo puede usar letras, números y espacios.

**3.** Elige una **categoría** para tu mundo:
- **Todos** — sin categoría específica
- **Naturaleza** — mundos con árboles, agua, naturaleza
- **Construcción** — edificios, castillos, estructuras
- **Mecanismo** — máquinas, inventos, cosas especiales

**4.** Toca **"Start Building"** — ¡y entra al editor!

## 4. El editor 3D

El editor es donde construyes tu mundo. Aquí es donde ocurre la magia. 🪄

### ¿Qué hay en el editor?

```
┌─────────────────────────────┐
│ ←    Grid  ○    SAVE        │  ← barra superior
├─────────────────────────────┤
│                          [+]│
│                          [2]│  ← control de nivel
│     GRID ISOMÉTRICO      [-]│
│       (aquí construyes)     │
│                             │
├─────────────────────────────┤
│ ↩ Deshacer    ↪ Rehacer    │  ← botones de acción
├─────────────────────────────┤
│ Estructura Decoración ...   │  ← categorías de bloques
├─────────────────────────────┤
│ 🪨  🪵  🧱  🟤  🟡  💎    │  ← barra de bloques
└─────────────────────────────┘
```

### Cómo colocar bloques

**1.** En la barra inferior toca la **categoría** de bloque que quieres:
- **Estructura** — piedra, madera, ladrillo, metal
- **Decoración** — horno, mesa de trabajo, cristal, lana
- **Naturaleza** — tierra, nieve, hojas, agua, lava
- **Especiales** — oro, diamantes, cofres, minerales

**2.** Toca el **bloque específico** que quieres usar — se resaltará con un borde morado

**3.** Toca cualquier **celda del grid** para colocar el bloque ahí

**4.** ¡Listo! El bloque aparece en esa posición

> 💡 **Tip:** Si tocas un bloque que ya colocaste, lo eliminas.

### Apilar bloques en altura

¿Quieres poner bloques uno encima del otro? Usa los botones **+** y **-** del lado derecho:

- Toca **[+]** para subir de nivel — el número del medio muestra en qué nivel estás
- Toca **[-]** para bajar de nivel
- El nivel **0** es el suelo, el nivel **9** es el más alto

**Ejemplo para hacer un árbol:**
1. Pon nivel en **0** → coloca troncos en la celda central
2. Sube a nivel **1** → coloca más troncos
3. Sube a nivel **2** → coloca hojas en las 4 celdas alrededor del tronco
4. Sube a nivel **3** → coloca hojas en la celda del tronco y alrededor

### Mover la cámara

| Gesto | Qué hace |
|---|---|
| **Deslizar con 1 dedo** | Mueve la cámara para ver otras partes del grid |
| **Pellizcar con 2 dedos** | Hace zoom — acerca o aleja la vista |
| **Rotar con 2 dedos** | Gira el escenario para verlo desde otro ángulo |

### Deshacer y rehacer

- Toca **"↩ Deshacer"** para quitar el último bloque que pusiste
- Toca **"↪ Rehacer"** para volver a poner el bloque que deshiciste

### La cuadrícula

Toca el **switch "Grid"** en la barra superior para activar o desactivar las líneas del grid. La cuadrícula ayuda a colocar bloques de forma más ordenada y precisa.

## 5. Guardar tu construcción

> ⚠️ **Importante:** La app NO guarda automáticamente mientras construyes. ¡Recuerda tocar SAVE!

### Cómo guardar

Toca el botón **"SAVE"** en la esquina superior derecha del editor.

Cuando se guarda correctamente verás un mensaje de confirmación en la pantalla.

### ¿Cuándo guardar?

- Cada vez que termines una sesión de construcción
- Antes de salir del editor
- Después de hacer cambios importantes

> 💡 **Tip:** Guarda frecuentemente para no perder tu trabajo.

---

## 6. Cargar un mundo guardado

### Desde la pantalla principal

Toca la **flecha →** en cualquier tarjeta de proyecto para ir directo al editor con ese mundo.

O toca **"CARGAR"** para ver la lista completa.

### Desde Cargar Mundo

Aquí verás todos tus proyectos guardados. Cada tarjeta muestra:
- 📋 Nombre del proyecto
- 📅 Fecha y hora de la última edición
- 🏷️ Categoría del mundo

**Para abrir un proyecto** toca la flecha **→** en la tarjeta.

**Para editar el nombre** toca **"CARGAR MUNDO"** — se abre el formulario con los datos para editar.

**Para eliminar** toca **"ELIMINAR"** → confirma en el diálogo → el proyecto se borra. Aparece un botón **"Deshacer"** por si te arrepientes.

### Buscar un proyecto

Usa la **barra de búsqueda** en la parte superior para filtrar proyectos por nombre.

## 8. Problemas comunes y soluciones

### 😴 La aplicación está lenta al abrir

**¿Qué pasa?** La app tarda 2-3 segundos en cargar la primera vez.

**¿Por qué?** La app necesita conectarse a Firebase para verificar tu sesión. Es normal.

**Solución:**
- Espera unos segundos — es solo la primera vez
- Si tarda más de 10 segundos, cierra la app y ábrela de nuevo
- Asegúrate de tener suficiente espacio libre en tu celular

---

### 💾 Mis bloques no se guardaron

**¿Qué pasa?** Entras a un proyecto y los bloques que pusiste antes no están.

**¿Por qué?** Saliste del editor sin tocar SAVE.

**Solución:**
- Siempre toca **SAVE** antes de salir del editor
- El botón SAVE está en la esquina superior derecha
- Después de guardar verás un mensaje de confirmación

---

### 🔐 No puedo iniciar sesión

**¿Qué pasa?** La app muestra un error al intentar entrar.

**Posibles causas y soluciones:**

| Problema | Solución |
|---|---|
| Contraseña incorrecta | Verifica que no tengas el Bloq Mayús activado |
| Correo incorrecto | Revisa que no tenga espacios o errores de escritura |
| Sin Internet | Firebase necesita Internet para el login — conéctate una vez |
| Cuenta no existe | Toca "Registrarse" para crear una cuenta nueva |

> 💡 **Tip:** Una vez que inicias sesión la app recuerda tu cuenta y no vuelves a necesitar Internet para usarla.

---

### 🎯 Los bloques no se colocan donde toco

**¿Qué pasa?** Tocas una celda del grid pero el bloque aparece en otro lugar.

**¿Por qué?** Esto puede pasar cuando has hecho mucho zoom o has rotado el escenario.

**Solución:**
- Vuelve a la vista normal — pellizca para ajustar el zoom
- El bloque se coloca exactamente donde toca el centro de tu dedo
- Si hiciste mucha rotación, regresa a la posición original rotando en sentido contrario
- El **highlight verde** te muestra exactamente dónde va a caer el bloque antes de soltar el dedo — úsalo como guía

## 🌟 Consejos para construir mejor

- **Planea antes de construir** — piensa en qué quieres hacer antes de empezar
- **Usa el nivel +/-** — los edificios necesitan altura, no solo superficie
- **Guarda frecuentemente** — toca SAVE cada vez que hagas cambios importantes
- **Usa la cuadrícula** — activa el Grid para colocar bloques más precisamente
- **Explora el inventario** — tienes 40 tipos de bloques diferentes para usar
- **Rota la cámara** — gira el escenario con 2 dedos para ver tu construcción desde todos los ángulos

## ❓ Preguntas frecuentes

**¿Necesito Internet para usar la app?**
Solo para iniciar sesión la primera vez. Después todo funciona sin Internet.

**¿Puedo tener varios mundos?**
¡Sí! Puedes crear tantos mundos como quieras. Todos se guardan en tu celular.

**¿Qué pasa si elimino la app?**
Perderás todos tus proyectos guardados. Antes de desinstalar guarda una captura de tus construcciones favoritas.

**¿Cuántos bloques puedo apilar?**
Hasta **10 bloques de altura** en cada celda del grid.

**¿El grid es siempre de 20x20?**
Sí, el grid siempre tiene 20 columnas y 20 filas — suficiente para construcciones grandes.
